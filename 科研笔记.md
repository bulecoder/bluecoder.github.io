# 科研笔记
## Segmentation方向的工作
1. MedNeXt (MICCAI 2023)
创新点： 作者认为传统的 U-Net 提取特征能力有限，于是他们借鉴了 Transformer 的扩展原则，设计了一个完全基于 ConvNeXt 模块的全新 3D 架构。

工程实现： 这篇论文的官方代码（nnunet_mednext）直接建立在 nnU-Net 框架之上。他们完全保留了 nnU-Net 强大的数据规划（Plan）和预处理（Preprocess）流程，仅仅在实例化模型的地方，把默认的 U-Net 替换成了他们自己的 MedNeXt 架构。


2. U-Mamba (MICCAI 2024)
创新点： 状态空间模型（SSM，尤其是 Mamba）是继 Transformer 之后深度学习界最火热的方向。U-Mamba 巧妙地将 Mamba 模块融入到网络中，用来处理医学图像中极长距离的全局依赖关系。

工程实现： 它的开源代码同样是原生嵌套在 nnU-Net 框架里的。这证明了即使是最前沿的序列模型，也可以完美兼容 nnU-Net 的动态 Patch 输入机制。


3. nnFormer (IEEE TMI 2022)
创新点： 这是一个非常经典的将 Transformer 引入 3D 医学图像的早期开创性工作。它结合了交错的卷积与自注意力操作，学习局部和全局的体积特征。

工程实现： 论文的名字就直白地告诉了大家，它是 "nnU-Net" 和 "Transformer" 的结合体。它不仅借用了代码，还在设计理念上深度借鉴了 nnU-Net 的优势。

4. 终极参考库：TriALS Challenge (MICCAI 2024/2025)
如果你去 GitHub 上搜索 TriALS 这个项目，你会发现一个巨大的宝藏。这是 MICCAI 挑战赛官方提供的一个标准化基准库，它的核心底层就是 nnU-Net V2。

更棒的是，这个仓库的作者已经提前帮你把 Swin-UNETR、LightM-UNet、U-Mamba 和 MedNeXt 等所有热门的改进模型都融合进 nnU-Net 里面了！



## Landmark detection方向的工作
对于纯landmark detection的工作：
1.  最新 SOTA：H3DE-Net (2025 年发布)
核心理念： 这是非常新的一个混合架构模型。它专门为了解决 3D 医学图像中特征点稀疏、信噪比低（比如 CBCT 中的金属伪影或散射）的问题，将 CNN（提取局部特征）和轻量级 Transformer（捕捉全局上下文）结合在了一起。
为什么适合你： 它是纯粹为 3D 关键点检测打造的，采用了多尺度特征融合，对于回归极其微小的特征点坐标精度极高。
Github 搜索/地址： ECNUACRush/H3DE-Net

2. 热力图回归鼻祖：Spatial Configuration-Net (SCN)；christianpayer/MedicalDataAugmentationTool-VerSe
由于原版的 SCN 是通用的 Landmark 代码，后来作者用 SCN 参加了 MICCAI 的大规模 3D 脊椎定位与分割挑战赛（VerSe 2019/2020）并拿了冠军。他把这套专门针对 3D 医疗影像的代码也开源了：
GitHub 仓库名： christianpayer/MedicalDataAugmentationTool-VerSe
说明： 这套代码展示了如何处理真实的 3D 医学影像（NIfTI 格式），如何先用一个 U-Net 找到大体位置，再用 SCN (SpatialConfiguration-Net) 回归热力图来精准定位特征点（脊椎中心点）。由于它直接处理 3D 坐标，这个仓库的代码逻辑对你的 CBCT 12个特征点检测任务参考价值极大！

3. 结合 3D 几何约束的最新力作：GeoRefineNet (ACCV 2024)
GitHub 仓库： Thanaporn09/GeoRefineNet
核心亮点： 传统的网络都是在瞎猜点，而这篇 2024 年底发表的工作专门针对 CBCT 数据，引入了强烈的 3D 几何结构约束。
契合你的创新点： 你的 12 个点中包含绝对对称的骨骼点（ZL / ZR 左/右颧骨点）以及相邻的牙齿点（M5 / M7）。GeoRefineNet 的多阶段细化模块能够利用这种先天几何信息。哪怕 M5R（右侧第二前磨牙）因为金属伪影看不清，它也能根据对侧的 M5L 和上方的颌骨点推算出来。你可以直接用它做 Baseline，然后在它的几何损失（Geometric Loss）上做微小改动就能出创新。

4. 解决超大 CBCT 尺寸的全局-局部框架：Maxillofacial-Landmark
GitHub 仓库： HLZ-CODE/Maxillofacial-Landmark
核心亮点： 专门针对口腔颌面部 CT / CBCT 扫描设计的“三阶段、由粗到细 (Coarse-to-Fine)”的特征融合框架。
契合你的创新点： 跑 CBCT 最大的痛点是显存爆炸。这套开源代码实现了一个极其优雅的流程：先用一个轻量级模块在下采样图像上抓取全局特征（比如确定 N 点和 PNS 点的大致上下颌位置），再在原始高分辨率下裁剪出局部 Patch 提取局部注意力特征。它的工程已经封装得很好，你可以在它的 UNet3D 骨干网络中替换成你想尝试的最新的 Transformer 模块。

5. 抗噪与特征对齐：CeLDA (MICCAI 2024)
GitHub 仓库： ShanghaiTech-IMPACT/CeLDA
核心亮点： 上海科技大学团队在 MICCAI 2024 发表的 SOTA 成果（Cephalometric Landmark Detection across Ages with Prototypical Network）。
契合你的创新点： 虽然它主要针对跨年龄段的特征变化，但其引入的“原型网络 (Prototypical Network)”思想非常前沿。对于 ANS（前鼻棘）这种在不同患者身上长得千奇百怪、极其容易被误判的点，利用它的原型特征对齐方法能极大提升鲁棒性。代码非常新，用的是 PyTorch，复现体验极度舒适。

6. 2024 MICCAI 挑战赛顶流方案：OptimisingfortheUnknown
GitHub 仓库： Julian-Wyatt/OptimisingfortheUnknown

核心亮点： 这是 2024 年 MICCAI 官方 Cephalometric Landmark Detection 挑战赛上，在 MRE（平均径向误差）指标上排名第一的开源方案。

契合你的创新点： 这个仓库最大的宝藏在于它的数据增强代码。在预测 M5 和 M7 等牙齿点时，患者牙齿上的正畸托槽或补牙材料在 CBCT 中会产生极强的放射状金属伪影。这个仓库里专门写了一套针对 X-ray/CBCT 伪影的增强程序（Artefact Augmentation）。你可以直接把这套增强逻辑“偷”过来，加到你的数据加载器里。
