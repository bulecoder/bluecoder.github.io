# 科研笔记
## Segmentation 方向的工作
1. MedNeXt (MICCAI 2023)
    * 论文题目：MedNeXt: Transformer-Driven Scaling of ConvNets for Medical Image Segmentation
    * 核心亮点：借鉴 Transformer 设计原则（如倒置瓶颈结构、大卷积核），设计了完全基于 ConvNeXt 模块的全新 3D 架构，旨在数据稀缺的医疗场景中实现长距离空间依赖建模。
    * GitHub: https://github.com/MIC-DKFZ/MedNeXt(工程实现：官方代码直接建立在 nnU-Net 框架之上，保留了强大的数据规划和预处理流程，仅在模型实例化阶段替换为 MedNeXt 架构。)

2. U-Mamba (MICCAI 2024)
    * 论文题目：U-Mamba: Enhancing Long-range Dependency for Biomedical Image Segmentation
    * 核心亮点：将状态空间模型（Mamba/SSM）融入网络，利用其线性计算复杂度实现长距离依赖建模，克服了 Transformer 处理 3D 高分辨率数据时显存爆炸的问题。
    * GitHub: https://github.com/bowang-lab/U-Mamba(工程实现：原生嵌套在 nnU-Net 框架内，完美兼容 nnU-Net 的自配置（Self-configuring）机制和动态 Patch 输入。)

3. nnFormer (IEEE TMI 2022)
    * 论文题目：nnFormer: Interleaved Transformer for Volumetric Segmentation
    * 核心亮点：将 Transformer 与卷积操作交错结合，通过自注意力机制学习全局特征，同时利用卷积保持局部细节，是 3D 医学影像 Transformer 的开创性工作。
    * GitHub: https://github.com/282857341/nnFormer(工程实现：深度借鉴 nnU-Net 的设计理念并重用其预处理框架，是 nnU-Net 与 Transformer 结合的典范。)

4. Swin UNETR (MICCAI 2021)
    * 论文题目：Swin UNETR: Swin Transformers for Semantic Segmentation of Brain Tumors in MRI Images
    * 核心亮点：将分层式的 Swin Transformer 作为 Encoder，通过平移窗口注意力机制捕获多尺度空间特征，在大型医学图像任务上表现强悍。
    * GitHub: https://github.com/Project-MONAI/research-contributions/tree/main/SwinUNETR

5. 标准化基准库：TriALS Challenge (MICCAI 2024/2025)
    * 核心价值：MICCAI 挑战赛官方提供的标准化基准库，底层为 nnU-Net V2，集成了 Swin-UNETR、LightM-UNet、U-Mamba 和 MedNeXt 等所有热门模型。
    * GitHub: https://github.com/Project-MONAI/TriALS

## Landmark Detection 方向的工作
1. 最新 SOTA：H3DE-Net (BIBM 2025)
    * 论文题目：H3DE-Net: A Hierarchical 3D Ensemble Network for Cephalometric Landmark Detection
    * 核心亮点：针对 3D 医学影像关键点稀疏、信噪比低的痛点，融合了 CNN 提取局部特征与轻量级 Transformer 捕捉全局上下文，回归精度极高。
    * 在公开3D CT数据集上实验，效果很好
    * GitHub: https://github.com/ECNUACRush/H3DE-Net

2. 热力图回归鼻祖：SCN (MedIA 2019)
    * 论文题目：Integrating Spatial Configuration into Heatmap Regression based CNNs for Landmark Localization
    * 核心亮点：正式提出了空间配置网络（SCN），通过两分支分别学习局部外观和全局点对点位置关系，是 3D 医疗关键点定位的必读经典。
    * 方法太老了
    * GitHub: https://github.com/christianpayer/MedicalDataAugmentationTool-HeatmapRegression

3. 几何约束 SOTA：GeoRefineNet (ACCV 2024)
    * 论文题目：GeoRefineNet: Geometry-aware Refinement Network for 3D Cephalometric Landmark Detection
    * 核心亮点：专门针对 CBCT 数据引入强 3D 几何结构约束，利用解剖学上的对称性（如左/右颧骨点）来纠正伪影导致的偏移。
    * 输入网络的是2D投影，可以使用3D数据（CBCT）
    * GitHub: https://github.com/Thanaporn09/GeoRefineNet

4. 全局-局部框架：Maxillofacial-Landmark (IEEE TIM 2025)
    * 论文题目：Enhanced Landmark Detection in Oral and Maxillofacial CT and CBCT Scans: A Multistage Global–Local Integration Approach
    * 核心亮点：（AI推荐 Baseline） 采用三阶段由粗到细（Coarse-to-Fine）策略，通过在下采样图定位 ROI 再在高分辨率 Patch 细化，完美解决 CBCT 显存爆表问题。
    * 在内部数据集合CT和CBCT上实验，也在公开数据集MMLD上面实验
    * GitHub: https://github.com/HLZ-CODE/Maxillofacial-Landmark

5. 特征对齐 SOTA：CeLDA (MICCAI 2024)
    * 论文题目：CeLDA: Cephalometric Landmark Detection across Ages with Prototypical Network
    * 核心亮点：引入原型网络（Prototypical Network）思想，增强了模型对不同年龄、不同解剖变异关键点的特征对齐和识别鲁棒性。
    * 数据是2D X-ray，不适合
    * GitHub: https://github.com/ShanghaiTech-IMPACT/CeLDA

6. 冠军方案：Optimising for the Unknown (MICCAI 2024)
    * 论文题目：Optimising for the Unknown: Domain Alignment for Cephalometric Landmark Detection
    * 核心亮点：2024 年挑战赛冠军，其核心竞争力在于开源了极强的抗伪影数据增强程序（Artefact Augmentation），对正畸托槽等金属干扰有极佳优化。
    * GitHub: https://github.com/Julian-Wyatt/OptimisingfortheUnknown


## 借用分割模型来做Landmark Detection
1. 基于大感受野（Transformer/混合架构）的CBCT关键点定位
    * 论文 1 (强烈推荐，完美对口)： Cephalometric Landmark Detection Using Graph- and Image-Aware Network with Landmark Contrastive Regularization (IEEE ISBI 2025)
    * 核心思想： 这是一篇极其纯正的 3D CBCT 特征点检测的最新 SOTA！作者提出了 **GrImNet（AI推荐的baseline）**。它不仅用 CNN 提取 3D 图像特征，还引入了图卷积网络（GCN）来处理由 CBCT 生成的三维骨骼模型的几何信息。这种图像+图拓扑的双重约束，对于你那 12 个具有强对称关联的解剖点来说，简直是量身定制。
    * 数据集需要CBCT、landmark 标注和 segmentation mask；野鸡文章，实验不足，参考文献少；私有数据集
    * GitHub: [github.com/RPIDIAL/GrImNet](https://github.com/RPIDIAL/GrImNet)


2. 拥抱 Mamba 的最新混合架构 (2024-2026)
    * 论文2: nnMamba: 3D Biomedical Image Segmentation, Classification and Landmark Detection with State Space Model (2024)  
    * 核心价值: 这个仓库的结构非常清晰，作者在主程序中专门预留并开源了 Landmark Detection 的训练分支（参考其仓库中的 nnMamba.py 脚本）。它是你学习如何将 Mamba 模块与坐标回归任务结合的绝佳参考。
    * GitHub: https://github.com/lhaof/nnMamba
    * 论文3:U-Mamba : Enhancing Long-range Dependency for Biomedical Image Segmentation (2024)
    * 核心价值: 这是多伦多大学马骏团队的官方神库，后续的 U-Mamba2 等牙科应用文章均是基于此底层框架修改而来。该代码库最大的优势是完美嵌套在 nnU-Net v2 的框架内，你完全可以利用 nnU-Net 的强大预处理能力，只抽取它的 Mamba 编码器模块进行改造。
    * GitHub: https://github.com/bowang-lab/U-Mamba
    

3. 域对齐与抗伪影数据增强 (Domain Alignment)
    * 论文: Optimising for the Unknown: Domain Alignment for Cephalometric Landmark Detection (MICCAI 2024)
    * 核心价值: 这是 MICCAI 2024 头影测量挑战赛的顶级方案。如果你发现你的模型在遇到带有金属牙套的 CBCT 时定位经常“乱跑”，直接去这个仓库里扒它的 数据增强（Artefact Augmentation） 源码，这套代码专门针对放射线图像的伪影干扰做了极强的鲁棒性优化。
    * GitHub: https://github.com/Julian-Wyatt/OptimisingfortheUnknown
